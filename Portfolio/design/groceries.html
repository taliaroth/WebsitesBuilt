Due to the outbreak of COVID-19, many of us have limited access to our usual groceries. Therefor, I created a search engine using Spoonacular's recipe API. This website allows users to search for the ingredients they want to include, and they'll have access to a list of recipes that use those ingredients.

For more information on how it was coded, check out my GitHub page!
